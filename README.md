**Just a simple example of MERN app.**
1. Clone the repository,
2. Run 'npm install'.
3. Use the following commands:
    - run app: 'gulp'
    - run app in 'watch' mode: 'gulp watch'
    - view logs: 'pm2 logs'
    - run tests: 'gulp mocha'
    
---
**NOTE**

You must have an installed MongoDB
